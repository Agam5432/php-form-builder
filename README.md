# PHP Form Builder

This project is a dynamic PHP Form Builder where an admin can create forms and users can submit responses.

## Features
- Create and manage forms
- Add multiple form fields dynamically
- Store form submissions in database
- View submitted data in admin panel

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Installation Steps

1. Download the project.
2. Copy the project folder to:
   C:\xampp\htdocs\
3. Start Apache and MySQL in XAMPP.
4. Create a database in phpMyAdmin.
5. Import the SQL file provided.
6. Update database credentials in:

config/database.php

7. Open the project in browser:

http://localhost/php-form-builder
